# PopInDashboard

PopInDashboard is a data analysis and visualization project that explores Meetup event data to identify patterns in event categories, locations, and attendee participation. The project uses Python, Google Colab, and Streamlit to transform raw event datasets into an interactive dashboard for easier exploration and insights.

Tools & Technologies
- Python
- Pandas for data processing
- Streamlit for interactive dashboard visualization
- Google Colab for development and execution
- GitHub for project version control

Dataset
The project uses two datasets:
- MeetUp_PopIn_Events.csv – contains information about Meetup events.
- Attendees_Dataset.csv – contains attendee participation data.
These datasets are cleaned and analyzed to uncover meaningful trends in event engagement.

Project Features
- Data cleaning and preprocessing using Python and Pandas
- Analysis of event categories and participation trends
- Visualization of event activity using interactive charts and dashboards
- Integration of multiple datasets to better understand event attendance patterns

Key Insights
The dashboard helps explore insights such as:
- Most popular event categories
- Locations with the highest event activity
- Patterns in attendee participation
- Trends in event engagement over time

How to Run the Project
- The Streamlit dashboard can be executed using Google Colab.
- Open the PopIn_GC.ipynb notebook in Google Colab.
- Run the notebook cells to load and process the datasets.
- In the final code cell, run the Streamlit application.
- A public tunnel link will be generated, allowing access to the dashboard in a browser.

Project Purpose
This project demonstrates practical data analysis and visualization skills, including data cleaning, dataset integration, and dashboard creation using Python and Streamlit. It serves as part of a data analytics portfolio project.
